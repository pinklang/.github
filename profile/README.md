# Pink

Flexible programming language; runs on a dynamic JVM-like VM

#### Goals
* Ease of use
* Remove complexity
* Fast development speeds

#### Example

```red
main: fn [args] [
	print "Does the dragon bite? "
	for-each args arg [
		if arg = "yes" [
			print "Scary"
		]
	]
]
```
